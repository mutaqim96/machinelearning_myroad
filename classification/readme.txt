Mengandungi source code untuk buat pelbagai classification type model.
Nak selesaikan masalah classification.

Kelebihan dan Kekurangan setiap model
==Logistic Regression==
+Probabilistic approach, beri info pasal statistical signigicance sesebuah feature.
- Logistic Regression Assumptions.

==KNN==
+ Senang faham, cepat dan efficient
- Kene pilih berapa bilangan neighbours K

==SVM==
+ Performant(function seperti expected), tak biased disebabkan outliers, tak mudah overfitting.
- Tak sesuai untuk masalah non-linear problems, tak sesuai untuk data yang ada banyak features.

==Kernel SVM==
+ High performance pada non-linear problems, tak biased disebabkan outliers, x sensitive kepada overfitting
- bukan pilihan terbaik untuk data yang ada banyak features, 
- lebih complex

==Decision Tree Classification==
+ ada Interpretablitiy, 
+ x perlu feature scaling, 
+ boleh digunakan pada masalah L dan NL
- Kalau dataset kecik result dia teruk, 
- overfitting senang jadi

==Random Forest Classification==
+ Powerful and accurate
+ Perfomr kat macam2 jenis masalah termasuk NL
- Xde interpretability
- senang untuk overfit, perlu pilih bilangan trees
