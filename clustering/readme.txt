2 type of clustering available within classification problem.
> K-Means Clustering
> Hierarchical Clustering

Kelebihan dan Kekurangan :

==K-Means==
+ senang nak faham
+ senang nak adapt
+ bagus untuk dataset kecil atau besar
+ laju, efficient dan x kecewakan
-  perlu tentukan bilangan cluster.

==Hierarchical Clustering==
+ Bilangan cluster yang optimal(yang tricky kat kmeans tadi), boleh didapati daripada model tu sendiri
+ visualization dia practical sebab guna dendrogram
- tak sesuai untuk large dataset.
